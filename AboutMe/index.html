About Parvez Rafi

And he dreams to build applications that would be able transform societies across the globe, not just priviledged sections. This belief motivates him to learn about different machine learning algorithms and implement them. Parvez developed this website with the aim of sharing his learning curve with the world. Afterall as it goes without saying… Knowledge increases by sharing.
